　在CatalogApi專案的資料夾Controllers，新增CatalogController類別，並繼承ControllerBase，也增加屬性[Route(“api/[controller]“)]與[ApiController]。

　　此Action為Http Post，在CatalogItems新增CatalogItem物件，並回傳Status Code 201
