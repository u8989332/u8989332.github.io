[讀書筆記] Professional JavaScript for Web Developers - 第1章 JavaScript簡介

Posted on 2020-05-10 Edited on 2023-11-19 In JavaScript , 讀書筆記

1.1 JavaScript簡史

  • European Computer Manufacturers Association (ECMA)專門制定JS
  • ECMA-262為ECMAScript (發音 ek-ma-script)
  • ISO將ECMAScript採納為ISO/IEC-16262, 瀏覽器開發商以此為標準實作JS的功能

1.2 JavaScript實作

  • JS包含三大元素
  1. 核心 (ECMAScript)
  2. Document Object Model (DOM)
  3. Browser Object Model (BOM)
  • ECMAScript本身和瀏覽器無直接關係
  • 瀏覽器只是Host environment, 可以執行ECMAScript的平台, 可以和DOM交互作用
  • 其他Host environment包含Node.js、過時的Adobe Flash
  • ECMAScript定7大種功能
  1. Syntax
  2. Types
  3. Statements
  4. Keywords
  5. Reversed words
  6. Operators
  7. Global objects

1.2.1 ECMAScript

1.2.1.1 ECMAScript的版本

  • ECMAScript 3.1: 2009發布的第5版修改(ES5), 包含JSON解析與序列化、method的繼承等
  • ECMAScript 6: 2015第6版修改, 也稱為ES6、ES2015
  • ECMAScript 7: 2016第7版修改, 也稱為ES7、ES2016
  • ECMAScript 8: 2017第8版修改, 也稱為ES8、ES2017

1.2.1.2 ECMAScript的兼容性

  • 必須支援ECMAScript規範的types, values, objects, properties, functions, program syntax, program semantics
  • 必須支援 Unicode字元集
  • 可擴展額外的types, values, objects, properties, function
  • 支援沒在ECMAScript定義的正規表達式語法

1.2.1.3 瀏覽器對ECMAScript的支援

  • IE 8不完整支援ES3 (才ES3!?), IE 9也不完整支援ES5 (!?), 直到IE 10以後版本才完整支援ES5, IE真是前端工程師的惡夢

  • Edge 12+有支援ES6

  • 其他瀏覽器只要有更新, 別太擔心ES的版本啦~

 

1.2.2 Document Object Model

  • 對於XML/HTML文件的API
  • 每一層節點都可以剖析成樹狀圖
  • W3C規劃DOM的功能
  • 最早DOM1核心是以XML做Mapping, 後來延伸HTML的DOM
  • 不只是JS才有DOM, 其他程式語言也有實作
  • DOM2新增多個功能的Interfaces

  1. DOM View:文件變化(比如CSS套用)前/後的對照

  2. DOM Events: 事件處理

  3. DOM Style: 應用CSS

  4. DOM Traversal and Range: 操作文件樹

  • DOM3新增 文件的新增、載入(DOM Load & Save)、文件的驗證 (DOM Validation)

  • 其他的DOM標準像是支援SVG、MathML數學標記、SMIL(Synchronized Multimedia Integration Language)

  • DOM1~3也是有區分瀏覽器版本的支援(!!??)

1.2.3 Browser Object Model

  • BOM可以操作瀏覽器顯示頁面內文以外的東西

  • 最早的BOM沒有規範

  • HTML5發布, 才將BOM寫入規範

  • 常見擴展的功能

  1. 彈出新視窗

  2. 移動、縮放、關閉

  3. 瀏覽器詳細資訊的navigator物件

  4. 瀏覽器載入頁面的詳細資訊location物件

  5. 使用者視窗解析度screen物件

  6. 對cookie的支援

  7. XMLHttpRequest和IE獨特的ActiveXObject物件

1.3 JavaScript版本

  • 只有Mozilla (前身Netscape)才有沿用JS的版本, 但也只有到1.8.5

  • 歷史版本簡介: https://developer.mozilla.org/en-US/docs/Archive/Web/JavaScript/New_in_JavaScript

  • 現在大宗瀏覽器都以ECMAScript版本為主
